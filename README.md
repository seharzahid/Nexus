# Entrepreneur Investor Dashboard

A premium React & TypeScript dashboard application designed for entrepreneurs to manage startup metrics, track investor relations, and schedule interactive boardroom meetings.

## 🚀 Key Features Implemented

- **Data Persistence:** Integrated `localStorage` across all modules (timing slots, meeting history, pitch deck documents, and investor tracking logs) so no data is lost on page refresh.
- **Advanced Analytics:** Integrated responsive Line and Bar charts using the `recharts` library to display real-time profile growth trends and pitch deck download rates.
- **Interactive Boardroom:** Developed a live virtual boardroom component with simulated premium video call functionality.
- **Investor CRM Filters & Memos:** Built custom multi-dropdown industry/stage filters paired with dynamic, private memo input logging for each investor.
